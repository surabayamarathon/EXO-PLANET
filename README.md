# 🎤 EXO PLANET #6 – EXplOrizon Jakarta Landing Page

Landing page ini dibuat untuk kebutuhan pendaftaran tiket event **EXO PLANET #6 – EXplOrizon in Jakarta**.
Website ini menampilkan informasi event serta formulir pendaftaran peserta secara online.

* Pemilihan kategori tiket & metode pembayaran
